# 🚀 Featured Projects

### 🛰️ [Automation Dashboard using PIC18F4580](https://github.com/Tarun460208/Automation-Dashboard-MC--PIC18F4580-)
A real-time embedded monitoring and control system designed for intelligent industrial or automotive automation.
- **Key Features:** CAN & UART Communication, Fault Detection, Sensor Interfacing, and Dashboard Visualization.
- **Tech Stack:** Embedded C, PIC18F4580 Microcontroller.

### 🏦 [Banking Management System](https://github.com/Tarun460208/Banking-management-system)
An architectural-grade object-oriented banking system engineered fully from the ground up.
- **Key Features:** Full CRUD Operations, Binary File Persistence (`fstream`), and clean multi-file OOP architecture.
- **Tech Stack:** Pure C++ (Zero external databases or third-party libraries).

### 🔍 [File Word Inversion using Hashing](https://github.com/Tarun460208/File-Word-Inversion-using-Hashing-C---DSA-)
A high-performance inverted search indexing engine built for lightning-fast file parsing and querying.
- **Key Features:** Custom Hash Table implementation, Linked Lists for conflict resolution, and ultra-fast multi-file data retrieval.
- **Tech Stack:** C, Data Structures & Algorithms (DSA).

### 🌳 [Red Black Tree Implementation](https://github.com/Tarun460208/RED-BLACK-TREE)
An interactive, menu-driven self-balancing binary search tree conforming strictly to standard RBT rules.
- **Key Features:** Insertion, Deletion, Search, Min/Max tracking, Inorder Traversal, and live structural property validation using recoloring/rotations.
- **Tech Stack:** C.

### 🔒 [Steganography (Image + Audio)](https://github.com/Tarun460208/Steganography-IMAGE-AUDIO-)
A secure cryptographic utility designed to seamlessly conceal and extract confidential payloads within media files.
- **Key Features:** LSB (Least Significant Bit) manipulation, bitwise processing, robust header corruption handling, and asset integrity protection.
- **Tech Stack:** C, File I/O, Bitwise Operations.
